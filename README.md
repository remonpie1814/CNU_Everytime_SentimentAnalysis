# News_Site_SentimentAnalysis

뉴스 기사 감성분석 프로젝트

프로젝트 요약 :



1. 배경 
2021년 언론 수용자 실태 조사에 따르면 조사인구의 46%가 가짜 뉴스, 편파적 기사를 한국 언론의 가장 큰 문제점으로 꼽는다.  
언론의 편파성이 정말 존재할까? 어떤 분야에서 편파적인 기사가 날까?  
이를 검증해보기 위해 실제 데이터와 감성분석의 결과를 비교해보기로 했다.
